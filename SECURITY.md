# Security

## Reporting a vulnerability

Please report security issues **privately** so we can fix them before public disclosure.

**Preferred:** use [GitHub Security Advisories](https://github.com/razroo/ray/security/advisories/new) for this repository (Private vulnerability reporting).

If that is not available for your account, open a **draft** security email to the maintainers or contact Razroo through your existing support channel and reference this repository.

Please include:

- A short description of the issue and its impact
- Steps to reproduce (proof-of-concept if possible)
- Affected versions or commits if known

We aim to acknowledge reports within a few business days and coordinate a fix and release timeline with you before details are published.

## Scope

Reports should concern **this repository** (`ray`): gateway, libraries, scripts, and documented deployment paths. Issues in upstream dependencies should also be reported to those projects where appropriate.

## Preferred fixes

Security fixes are shipped through normal releases (semver as appropriate). Critical fixes may be backported or released as patch versions for supported release lines.
